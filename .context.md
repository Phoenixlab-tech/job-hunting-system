# Job Hunting System — Context

## Project Overview
Multi-agent job hunting system for Arnaud (6 professional profiles). Fetches job offers from France Travail API, matches them against profiles, and generates applications.

## Agent Architecture
- **Agent 1**: Job Fetcher → France Travail API → Google Sheets
- **Agent 2**: Job Matcher → Score offers against profiles
- **Agent 3**: Application Generator → Create customized cover letters
